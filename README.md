# azurazure openai VUE和node.js示例程序

## 功能

* Chat Completeion接口
* token计算接口

## 项目结构

* frontend VUE前端，默认端口8080
* backend node.js后端，默认端口8081
* lib共享库

## Project 配置

请到frontend和backend后边运行
```
npm install
```

### 运行

前端
```
npm run serve
```

后端
```
node server.js
```
